# Kaiserreich for HOI4 Issue Tracker

You can submit bug reports and suggestions for both the main mod and all the various submods here.

When submitting an issue, please make sure you select the correct template and read the instructions carefully.

[**> Submit a new issue**](https://github.com/Kaiserreich/Kaiserreich-4-Bug-Reports/issues/new/choose)

[**> Check existing issues**](https://github.com/Kaiserreich/Kaiserreich-4-Bug-Reports/issues)

For a step-by-step guide on how to create an issue, refer to [*Reporting Bugs and Making Suggestions*](https://kaiserreich.wiki/Reporting_Bugs_and_Making_Suggestions) on the Kaiserreich Wiki.

Please note that issues for [**KRDH**](https://github.com/Kaiserreich/Kaiserreich-DH) and the [**KRDH music pack**](https://github.com/Kaiserreich/Kaiserreich-DH-Music) are handled separately.

Thanks for all your help!

*The Kaiserreich team*
